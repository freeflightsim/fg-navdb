Navigation Data for FlightGear

This project takes the latest xplane data, imports it to a postgis spatial database ready for querying on demand. It is intended as a "permanent" feed moving foward and making the latest data available and updated regular. The data returned is formatted in various formats and in a practical way with a lot of the calculations already done, such as the middle of an airport, runway center point, orientation and displaced threshold etc. Ideal for mashups, et all invited.
